# 字符串构造

- 字符串可用双引号或者单引号包含，但是不可出现两个单引号或者两个双引号

# 字符串函数

### 大小写相关

####  title()

- 修改单词首字母大小写
- ”ada love“    ---->"Ada Love"

#### upper()

- 全部字母修改为大写

#### lower()

- 全部字母修改为小写

### 字符串分割合并相关

- 直接使用 + 号来合并
- \t  == tab
- \n == 换行
- 可以同时包含

#### rstrip()

- 删除字符串末尾的空白 （right）

#### lstrip()

- 删除字符串开头的空白  （left）

#### strip（）

- 删除字符串两端的空白
#### split()
- 以空格为分隔把字符串分割为列表
