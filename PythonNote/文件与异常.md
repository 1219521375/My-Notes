# 文件
- 'r' 只读模式
- 'w' 只写模式（覆盖）
- 'a' 附加模式（添加内容到末尾）
#### 打开文件
```
with open('文件路径') as 变量名:
```
- 使用该变量名操作文件

### 文件操作函数
#### read()
- 返回整个文件内容的字符串（末尾带一个换行）

#### readlines() && readline()
- 返回文件中每一行字符串组成的列表
- 返回一行字符串（一行行往下
- 每一行末尾均带有一个\n

#### write('写入内容')
```
with open('文件路径','w') as 变量名:
```
- 写入函数
- 与终端输出类似换行需要换行符\n
