# 关于Python中切片操作的理解

## a = [0, 1, 2, 3, 4, 5]

- a[ x : y ] 表示取数组的下标x到y（不包括y） 以下任何切片操作遵循这条规则

- x若为空，则表示数组首元素之前的元素 eg：a中0之前一位（假设存在）

- y若为空，则表示数组尾元素之后的元素 eg：a中5之后一位（假设存在）

- 负数表示末端开始数第几个，eg：a[ -1 ] = 5,  a[ -2 ] = 4

  ```python
  a = np.array([0, 1, 2, 3, 4])
  b = [0, 1, 2, 3, 4]
  print(b[1:3])
  print(b[4:])
  print(b[:0])
  print(b[-3:-1])
  
  结果为：
  [1, 2]
  [4]
  []
  [2, 3]
  
  ```
